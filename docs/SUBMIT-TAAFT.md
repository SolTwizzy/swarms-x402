## Short Description (for form)

SwarmX is a platform for AI agent tasks powered by x402 micropayments and Swarms multi-agent orchestration. It offers 47 HTTP endpoints for contract audits, token diligence, DeFi risk scoring, fact-checking, code audits, research reports, compliance analysis, and more. Pay per call with USDC ($0.001-$5) — no API keys, accounts, or subscriptions needed. The x402 protocol means payment IS the authentication. Also provides 39 MCP tools for AI agent integration and works as both a standalone API server and an ElizaOS v2 plugin. Free tier: 10 calls/day per IP.

## Long Description (for form)

SwarmX bridges two powerful systems: x402 micropayments (via the Dexter SDK) for seamless USDC payments, and Swarms multi-agent orchestration for complex AI tasks using 15+ swarm architectures.

**Key Features:**
- 47 HTTP endpoints across 7 categories (crypto analysis, trading signals, task orchestration, data extraction, wallet analysis, etc.)
- x402 protocol payments: no API keys, no accounts, no subscriptions — just pay per call with USDC on Solana or EVM
- 39 MCP tools for AI agent discovery and integration
- 15+ swarm architectures: research pipelines, analysis panels, code review, debate-and-decide
- Single-agent tasks route directly to OpenAI for cost efficiency (~$0.001/call)
- Multi-agent tasks use Swarms API for orchestrated analysis ($1-$5/call)
- Free tier: 10 calls/day per IP
- Dual distribution: standalone HTTP server (Railway/Docker) AND ElizaOS v2 plugin (npm)

**Tech Stack:** TypeScript, Bun, x402 protocol, Swarms API, OpenAI, ElizaOS v2
