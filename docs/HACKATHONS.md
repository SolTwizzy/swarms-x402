## SUBMISSION TEMPLATE (Reusable)

### Project Name
SwarmX

### One-liner
AI Agent Teams. One Payment. Multi-agent orchestration with x402 micropayments.

### Description
SwarmX is a production-ready platform that combines x402 micropayments (Coinbase/Dexter) with Swarms multi-agent orchestration to enable AI agents to autonomously buy and sell services over HTTP. Agents pay each other per-call using USDC via the x402 protocol — no API keys, no accounts, just standard HTTP 402 responses.

### Key Features
- 28 x402-protected API endpoints (crypto analysis, wallet tracking, trading signals, multi-agent tasks)
- 15+ swarm architectures (research pipelines, analysis panels, code review, debate)
- Dual deployment: standalone HTTP server OR ElizaOS v2 plugin
- Access pass tiers for high-frequency endpoints (1h/24h/7d/30d)
- Payment persistence with Drizzle ORM
- Free tier (10 calls/day) with automatic x402 upgrade
- Live on Solana mainnet with real USDC transactions

### Tech Stack
- TypeScript, Bun runtime
- x402 via @dexterai/x402 (Dexter SDK)
- Swarms API for multi-agent orchestration
- OpenAI for single-agent tasks (cost optimization)
- ElizaOS v2 compatible
- Deployed on Railway (https://api.swarmx.io)
